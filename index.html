Scorekeeper

This is a project to illustrate the workings of eventhandlers in Javascript.

The goal of this project is to have 2 teams, playing a sport of choice with each one having a button to incrementing the score.

There should be a textfield which shows the results on the fly when the user presses a button.

The first team that scores 5 points is the winner. In that case, the following happens

  • The buttons become unresponsive
  • The winner gets announced by a label below the score
